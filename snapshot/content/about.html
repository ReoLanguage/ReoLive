About

More details about this project can be found in GitHub:
  • https://github.com/ReoLanguage/ReoLive - Project that produces the JavaScript code used this website.
  • https://github.com/ReoLanguage/Preo - Project that parses and analyses connectors in the Preo language.
  • https://github.com/arcalab/hybrid-programming - Project that parses and analyses hybrid programs.
  • https://github.com/arcalab/hubAutomata - Project that parses and analyses Hub Automata in the context of VirtuosoNext.
  • https://github.com/haslab/ifta - Project that parses and analyses families of (real-time) automata.
